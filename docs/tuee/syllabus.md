# TUEE Syllabus

Comprehensive syllabus for the TUEE MCA entrance examination:

## Exam Pattern
* **Mode:** Computer Based Test (CBT)
* **Questions:** 50
* **Marks per question:** 2
* **Negative Marking:** 0.5
* **Duration:** 90 minutes

## Syllabus Topics
1. **Logical Reasoning and Analysis**
2. **Basic Mathematical Ability (10+2 Level Mathematics)**
3. **Programming and Problem Solving:**
    - Number System
    - Boolean Algebra
    - Logic Gates
4. **C Programming**
5. **English Vocabulary and Composition**
